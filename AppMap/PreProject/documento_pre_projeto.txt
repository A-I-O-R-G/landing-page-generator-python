O script deve interpretar a descrição, identificando os elementos cruciais a serem incluídos na landing page, tais como: 1. Título, 2. Subtítulo, 3. Imagens, 4. Textos descritivos, 5. Call-to-action (chamada para ação), e 6. Formulários de inscrição ou coleta de dados. O objetivo é que, ao receber a descrição como entrada, o script produza um código HTML/CSS pronto para uso, que possa ser facilmente integrado em um site ou plataforma de marketing online. Além disso, o script deve garantir que a landing page seja responsiva e otimizável para diferentes dispositivos e navegadores, visando oferecer uma experiência de usuário agradável e eficaz para conversão de visitantes em leads."}
