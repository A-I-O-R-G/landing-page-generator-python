
{"doc_Pre_Projeto": "O usuário está solicitando a criação de um script que possa gerar uma landing page (página de captura) de forma automatizada, utilizando uma descrição fornecida como entrada. O script deverá analisar a descrição e extrair elementos essenciais que devem estar presentes na landing page, tais como: 1. Título principal, 2. Subtítulos, 3. Parágrafos explicativos, 4. Imagens relevantes, 5. Chamadas para ação (call-to-action), 6. Botões de inscrição ou contato, e quaisquer outros elementos visuais ou funcionais necessários para capturar leads. O script também deverá gerar o código HTML e CSS necessário para a construção da página, assegurando que a landing page seja esteticamente atraente e responsiva, adaptando-se a diferentes tamanhos de tela e dispositivos. O objetivo é facilitar a criação de landing pages que possam ser rapidamente implementadas em campanhas de marketing digital, otimizadas para conversões e direcionadas ao público-alvo desejado."}
